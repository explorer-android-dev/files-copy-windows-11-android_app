# Файлы - Копия проводника Windows 11 для Android

Приложение "Файлы" - это полнофункциональный файловый менеджер для Android с интерфейсом в стиле Windows 11.

## 🚀 Особенности

### 🎨 Дизайн в стиле Windows 11
- **Fluent Design System** - современный интерфейс с акцентами Windows 11
- **Цветовая схема** - фирменный синий цвет Windows (#0078D4)
- **Анимации** - плавные переходы и свайп-обновление
- **Стилизованные иконки** - иконки типов файлов, похожие на Windows 11

### 📁 Файловый менеджер
- **Быстрая навигация** - боковая панель с доступом к основным папкам
- **История навигации** - кнопки "туда" и "обратно"
- **Хлебная крошка** - навигационная строка с путём
- **Поиск файлов** - быстрый поиск по имени файла
- **Сортировка** - по имени, дате, размеру
- **Операции с файлами**:
  - Копирование
  - Вырезание
  - Вставка
  - Удаление
  - Переименование
  - Создание папок
  - Просмотр свойств

### 👁️ Просмотр файлов
- **Список (Details)** - детальный вид с информацией о файлах
- **Значки (Icons)** - вид сеткой с большими иконками
- **Мультивыделение** - выделение нескольких файлов одновременно
- **SwipeRefresh** - обновление содержимого свайпом вниз

### 📁 Быстрый доступ (Боковая панель)
- **Этот ПК** - корневой каталог
- **Документы** - папка документов
- **Загрузки** - папка загрузок
- **Изображения** - папка изображений
- **Видео** - папка видео
- **Музыка** - папка музыки

### 📋 Буфер обмена
- Копирование файлов и папок
- Вырезание с удалением оригинала
- Вставка в целевую папку
- Рекурсивное копирование папок

## 🚀 Установка и сборка

### 1. Клонирование репозитория
```bash
git clone [repository-url]
cd files-copy-windows-11-android_app
```

### 2. Открытие в Android Studio
- Откройте Android Studio
- File → Open → выберите папку проекта
- Android Studio автоматически загрузит зависимости

### 3. Сборка APK
```bash
cd /path/to/project
./gradlew build
```

Готовый APK будет находиться в: `app/build/outputs/apk/debug/app-debug.apk`

### 4. Запуск на устройстве
```bash
./gradlew installDebug
```

## 📱 Испо
льзование

### Навигация
- **Кнопка Назад** - перейти на предыдущую папку
- **Кнопка Вперёд** - перейти на следующую папку (если была история)
- **Клик на хлебную крошку** - перейти в нужную папку
- **Клик на быстрый доступ** - открыть популярную папку

### Работа с файлами
- **Один клик** - открыть файл или папку (если включен режим выделения, то выделить)
- **Долгий клик** - включить режим выделения и показать контекстное меню
- **Контекстное меню**:
  - Копировать
  - Вырезать
  - Вставить
  - Удалить
  - Переименовать
  - Свойства

### Поиск
- **Кнопка поиска** - открыть диалог поиска
- Введите текст для поиска
- Результаты появляются в реальном времени

### Меню
- **Кнопка меню** - открыть основное меню
  - Новая папка
  - Вид: Детали
  - Вид: Значки
  - Сортировать по имени
  - Сортировать по дате
  - Сортировать по размеру
  - Выделить все

## 🔐 Разрешения

Приложение требует следующих разрешений:
- `READ_EXTERNAL_STORAGE` - чтение файлов
- `WRITE_EXTERNAL_STORAGE` - изменение файлов
- `MANAGE_EXTERNAL_STORAGE` - полный доступ (Android 11+)
