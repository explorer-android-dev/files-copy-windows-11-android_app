# Файлы - Копия проводника Windows 11 для Android

Приложение "Файлы" - это точная копия проводника Windows 11 (23H2), разработанная для Android. Приложение имеет интерфейс в стиле Fluent Design и предоставляет полный функционал для управления файлами.

## 🚀 Особенности

### 🎨 Дизайн в стиле Windows 11
- **Fluent Design** - современный интерфейс с акцентами Windows 11
- **Цветовая схема** - фирменные цвета Windows 11 (#0078D4)
- **Анимации** - плавные переходы и эффекты
- **Иконки** - стилизованные иконки проводника Windows 11

### 📁 Файловый менеджер
- **Навигация** - удобная навигация по папкам
- **Поиск файлов** - быстрый поиск по имени файла
- **Сортировка** - по имени, дате, размеру
- **Операции с файлами** - копирование, перемещение, удаление, переименование
- **Контекстное меню** - долгое нажатие для опций файла

### 📱 Оптимизация для Android
- **Разрешения** - автоматическое запрос необходимых разрешений
- **Адаптивный интерфейс** - поддержка разных размеров экранов
- **Производительность** - оптимизированная работа с большими папками

## 🛠️ Технический стек

## 🚀 Установка и сборка

### 1. Клонирование репозитория
```bash
git clone [repository-url]
cd "Андроид приложение - Файлы - Копия проводника 11 винды билда 23H2"
```

### 2. Открытие в Android Studio
1. Откройте Android Studio
2. Выберите "Open an existing project"
3. Выберите папку проекта

### 3. Синхронизация Gradle
1. Дождитесь синхронизации Gradle
2. При необходимости установите рекомендуемые SDK

### 4. Сборка проекта
1. Выберите устройство или эмулятор
2. Нажмите `Run` (Ctrl+R) или `Build` -> `Build Bundle(s) / APK(s)`

## 🔧 Настройка

### Разрешения
Приложение требует следующие разрешения:
- `READ_EXTERNAL_STORAGE` - чтение файлов
- `WRITE_EXTERNAL_STORAGE` - запись файлов  
- `MANAGE_EXTERNAL_STORAGE` - полный доступ к хранилищу (Android 11+)

### FileProvider
Приложение использует FileProvider для безопасного открытия файлов другими приложениями.

## 📱 Использование

### Основные функции:
1. **Навигация** - используйте кнопки "Назад", "Домой", "Обновить"
2. **Поиск** - нажмите на иконку поиска для поиска файлов
3. **Меню** - дополнительное меню для сортировки и создания папок
4. **Контекстное меню** - долгое нажатие на файл для опций

### Поддерживаемые типы файлов:
- 📄 Текстовые файлы (.txt, .log)
- 🖼️ Изображения (.jpg, .png, .gif, .bmp, .webp)
- 🎥 Видео (.mp4, .avi, .mkv, .mov, .wmv)
- 🎵 Аудио (.mp3, .wav, .flac, .aac, .ogg)
- 📋 PDF (.pdf)
- 📊 Microsoft Office (.doc, .docx, .xls, .xlsx, .ppt, .pptx)
- 📦 Архивы (.zip, .rar, .7z, .tar, .gz)
- 🤖 Android (.apk)

## 🐛 Известные проблемы

- На некоторых устройствах может потребоваться ручное предоставление разрешений
- Поиск в больших папках может занимать время
- Некоторые операции с файлами могут требовать дополнительных разрешений
